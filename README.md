## Requirement

- OpenCV3
- Eigen3
- ROS Melodic

## Install

~~~sh
mkdir -p dip_ws/src
cd dip_ws/src
git clone https://github.com/stdcat/DIP_Final_Project.git
cd ..
catkin_make
~~~

then just run 

~~~sh
rosrun dip_final dip_final
~~~
